# Writing

V1 is an IELTS Academic Writing Task 2 practice surface.

The writing flow is intentionally simple:

```text
Question -> Plan -> Write -> Evaluate -> Retry
```

V1 rules:

- deterministic evaluator only
- no chat or model-backed review
- all scoring language uses "Practice estimate"
- visible evidence must explain every score
- 40-minute coaching timer is guidance, not strict blocking
- local-first saved attempts
- detailed analysis stays behind "View detailed evaluation"

The evaluator checks:

- word count
- paragraph count
- repeated words
- missing conclusion
- unclear opinion
- weak linking
- sentence length and variety
- time spent, writing pace, and projected words at 40 minutes
